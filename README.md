
VeXeFy - GitHub-ready Android project (CI builds debug APK)

How to use:
1. Unzip this folder locally.
2. Push all files to an empty GitHub repository (do not add extra files that change structure).
3. Ensure the default branch is 'main' (or edit the workflow to match your branch).
4. After push, go to GitHub Actions → Build APK → wait for job to finish → download artifact.
